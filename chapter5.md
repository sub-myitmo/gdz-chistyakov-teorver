## Задача 1 (№2 из учебника)

Cлучайная величина ξ равномерно распределена на отрезке [0, 1]. Найти плотности распределения величин
а) ``` η1 = 2ξ + 1 ```;
б) ``` η2 = -ln(1-ξ) ```

### Решение:

Чтобы найти плотности распределения случайных величин ``` η₁ = 2ξ + 1 ``` и ``` η₂ = -ln(1 - ξ) ```, начнем с исходной случайной величины ξ, которая равномерно распределена на отрезке [0, 1]. Плотность распределения ξ имеет вид:

```
f(x) =
1) 0 ≤ x ≤ 1
2) 0, иначе
```

▎а) Для ``` η₁ = 2ξ + 1 ```

1. Найдем область значений: 
	Если ξ принимает значения от 0 до 1, то η₁ будет принимать значения от 2 ⋅ 0 + 1 = 1 до 2 ⋅ 1 + 1 = 3. Таким образом, η₁ распределена на отрезке [1, 3].

2. Найдем функцию распределения F_η₁(y):
- Для y < 1: ``` F_η₁(y) = 0 ```
- Для y ∈ [1, 3]:
	``` F_η₁(y) = P(η₁ ≤ y) = P(2ξ + 1 ≤ y) = P((ξ ≤ y - 1 / 2)) ```
	
	Поскольку ξ равномерно распределена: ``` F_η₁(y) = y - 1 / 2,   1 ≤ y ≤ 3 ```

- Для y > 3: ``` F_η₁(y) = 1 ```

3. Найдем плотность распределения f_η₁(y):

	Плотность f_η₁(y) является производной функции распределения:
	```
	f_η₁(y) = d / dy F_η₁(y) =
	1) 0, y < 1
	2) 1 / 2, 1 < y < 3
	3) 0, y > 3
	```


▎б) Для ``` η₂ = -ln(1 - ξ) ```

1. Найдем область значений: 
	
	Если ξ принимает значения от 0 до 1, то 1 - ξ принимает значения от 1 до 0, и следовательно, -ln(1 - ξ) принимает значения от 0 (при ξ = 1) до +∞ (при ξ = 0). Таким образом, η₂ распределена на отрезке [0, +∞).

2. Найдем функцию распределения F_η₂(y):

- Для y < 0: F_η₂(y) = 0
- Для y ≥ 0: ``` F_η₂(y) = P(η₂ ≤ y) = P(-ln(1 - ξ) ≤ y) ```
	
	Это эквивалентно: ``` -ln(1 - ξ) ≤ y ⇒ 1 - ξ ≤ e⁻ʸ ⇒ ξ ≥ 1 - e⁻ʸ ```

	Таким образом:
	```
	F_η₂(y) = P(ξ < 1 - e⁻ʸ) =
	1) 0, y < 0
	2) 1 - e⁻ʸ, y ≥ 0
	```

3. Найдем плотность распределения f_η₂(y):

	Плотность f_(η₂)(y) является производной функции распределения:
	```
	f_η₂(y) =
	1) 0, y < 0
	2) e⁻ʸ, y ≥ 0
	```

Таким образом, результаты для плотностей распределения:
```
f_η₁(y) =
1) 0,  y < 1
2) 1 / 2,  1 < y < 3
3) 0,  y > 3

f_η₂(y) =
1) 0,  y < 0
2) e⁻ʸ,  y ≥ 0

```

## Задача 2 (№6 из учебника)

Функция распределения F(x) величины ξ строго монотонна и непрерывна. Найти закон распределения величины η = F(ξ)

### Решение:

Чтобы найти закон распределения величины η = F(ξ), где F(x) — функция распределения случайной величины ξ, которая строго монотонна и непрерывна, мы можем воспользоваться следующим подходом.

▎Шаги для нахождения закона распределения η

1. Понимание функции F:

Поскольку F(x) — это функция распределения, она принимает значения в диапазоне [0, 1] и является строго монотонной и непрерывной. Это означает, что F(x) строго возрастает.

2. Находим распределение η:

Рассмотрим случайную величину η = F(ξ). Для нахождения функции распределения η нам нужно найти вероятность того, что η принимает значение меньше или равно y:

``` 
P(η ≤ y) = P(F(ξ) ≤ y) 
```


3. Обратное преобразование:

Поскольку F(x) строго монотонна и непрерывна, можно решить неравенство F(ξ) ≤ y относительно ξ: ``` ξ ≤ F⁻¹(y)```, где F⁻¹(y) — это обратная функция к F.

4. Запись вероятности:

``` 
P(F(ξ) ≤ y) = P(ξ ≤ F⁻¹(y)) 
```

Это выражение равно функции распределения случайной величины ξ:

``` 
P(ξ ≤ F⁻¹(y)) = F(F⁻¹(y)) = y 
```

для y в диапазоне [0, 1].

▎Заключение

Следовательно, закон распределения величины η = F(ξ) будет равномерным на отрезке [0, 1]:

``` 
F_η(y) = P(η ≤ y) = y,   0 ≤ y ≤ 1. 
```

Это значит, что случайная величина η имеет равномерное распределение на интервале [0, 1].

## Задача 3 (№13 из учебника)

Найти закон распределения суммы двух независимых случайных величин ξ1 и ξ2 из которых имеет распределение Пуассона с параметром λ1 и λ2 соответсвенно

### Решение:

Сумма двух независимых случайных величин, каждая из которых имеет распределение Пуассона, также имеет распределение Пуассона. 

Если ξ₁ и ξ₂ — независимые случайные величины с распределениями Пуассона с параметрами λ₁ и λ₂ соответственно, то сумма ξ = ξ₁ + ξ₂ будет иметь распределение Пуассона с параметром λ = λ₁ + λ₂.

▎Формально

Если ξ₁ ∼ Poisson(λ₁) и ξ₂ ∼ Poisson(λ₂), то:

``` ξ = ξ₁ + ξ₂ ∼ Poisson(λ₁ + λ₂) ```


▎Плотность вероятности

Плотность вероятности (или функция вероятности) для распределения Пуассона задаётся следующим образом:

``` 
P(ξ = k) = e^-λ λᵏ / k!,   k = 0, 1, 2, … 
```


Для суммы двух независимых случайных величин с распределением Пуассона мы можем записать:

``` 
P(ξ = k) = e^(-λ₁ - λ₂) (λ₁ + λ₂)ᵏ / k!,   k = 0, 1, 2, … 
```

Таким образом, закон распределения суммы двух независимых случайных величин с распределением Пуассона задаётся как распределение Пуассона с параметром λ₁ + λ₂.

## Задача 4 (№14 из учебника)

Обозначим τ число испытаний в схеме Бернулли до появления первого успеха включительно. Найти закон распределения τ

### Решение:

В схеме Бернулли, когда мы проводим испытания до появления первого успеха, случайная величина τ будет следовать геометрическому распределению.

▎Определение

Геометрическое распределение описывает число испытаний до первого успеха в последовательности независимых испытаний, где каждое испытание имеет вероятность успеха p и вероятность неудачи q = 1 - p.

▎Закон распределения

Случайная величина Θ имеет следующие характеристики:

Вероятность того, что первый успех произойдёт на k-ом испытании (где k = 1, 2, 3, …):
```
P(Θ = k) = p ⋅ qᵏ⁻¹,
где:
- p — вероятность успеха в одном испытании,
- q = 1 - p — вероятность неудачи в одном испытании.
```

▎Пример

Если, например, вероятность успеха p = 0.3, то:

P(Θ = k) = 0.3 ⋅ (0.7)ᵏ⁻¹,   k = 1, 2, 3, …


▎Ожидаемое значение и дисперсия

Для геометрического распределения также известны ожидаемое значение и дисперсия:

- Ожидаемое значение: ``` E[τ] = 1 / p ```
- Дисперсия: ``` D[τ] = q / p² = 1 - p / p² ```


Таким образом, случайная величина τ, представляющая число испытаний до первого успеха в схеме Бернулли, имеет геометрическое распределение с параметром p.

## Задача 5 (№16 из учебника)

Найти закон распределения величины τₘ равной числу испытаний в схеме Бернулли до появления m-го успеха

### Решение:

В схеме Бернулли, когда нас интересует количество испытаний до появления  m-го успеха, мы имеем дело с распределением негативного биномиального. Это распределение описывает количество неудач, которые происходят до достижения заданного числа успехов.

▎Определение:

Если  X  — случайная величина, обозначающая общее количество испытаний до достижения  m -го успеха, то её распределение описывается следующим образом:
```
P(X = k) = \binomk-1m-1 pᵐ (1-p)ᵏ⁻ᵐ,
где:
- k  — общее количество испытаний (где  k = m, m+1, m+2, … ),
- p  — вероятность успеха в каждом испытании,
- (1-p)  — вероятность неудачи,
- \binomk-1m-1  — биномиальный коэффициент, который показывает количество способов выбрать  m-1  успехов из первых  k-1  испытаний.
```
▎Пример:

Допустим, вероятность успеха  p = 0.3 , и мы хотим найти вероятность того, что для достижения второго успеха потребуется 5 испытаний ( m = 2 ,  k = 5 ):
```
P(X = 5) = \binom5-12-1 (0.3)² (0.7)⁵⁻² = \binom41 (0.3)² (0.7)³
```

Вычисляем:
```
P(X = 5) = 4 ⋅ 0.09 ⋅ 0.343 = 4 ⋅ 0.03087 ≈ 0.12348
```

▎Свойства:

• Математическое ожидание:  ``` E(τₘ) = m/p ```

• Дисперсия:  ``` D(τₘ) = (m(1-p))/p² ```

Таким образом, случайная величина  τₘ , равная числу испытаний до появления  m -го успеха, подчиняется негативному биномиальному распределению с параметрами  m  и  p .

## Задача 6 (№15 из учебника)

Величина τ₁ равна числу испытаний в схеме бернулли до первого успеха включительно, τ₂ - число испытаний, прошедших после первого успеха до второго успеха. Найти совместное распределение τ₁, τ₂. Являются ли τ₁ и τ₂ независимыми?

### Решение:

Для решения задачи о совместном распределении случайных величин τ₁  и  τ₂ , где  τ₁  — это число испытаний до первого успеха (включительно), а  τ₂  — число испытаний после первого успеха до второго успеха, мы можем использовать свойства распределений Бернулли и негативного биномиального распределения.

▎Распределение τ₁

Случайная величина  τ₁  подчиняется геометрическому распределению с параметром  p  (вероятностью успеха):
```
P(τ₁ = k) = (1-p)ᵏ⁻¹ p,   k = 1, 2, 3, …
```

▎Распределение τ₂

Случайная величина  τ₂  также подчиняется геометрическому распределению с тем же параметром  p . Она представляет собой количество испытаний до первого успеха после того, как первый успех уже произошел. Таким образом, её распределение также будет:
```
P(τ₂ = j) = (1-p)ʲ⁻¹ p,   j = 1, 2, 3, …
```

▎Совместное распределение (τ₁, τ₂)

Так как  τ₁  и  τ₂  являются независимыми геометрическими распределениями, их совместное распределение можно выразить как произведение их индивидуальных распределений:
```
P(τ₁ = k, τ₂ = j) = P(τ₁ = k) P(τ₂ = j) = (1-p)ᵏ⁻¹ p ⋅ (1-p)ʲ⁻¹ p = p² (1-p)⁽ᵏ⁺ʲ⁻²⁾, где  k ≥ 1  и  j ≥ 1 .
```

▎Независимость

Случайные величины  τ₁  и  τ₂  являются независимыми, так как результат одного испытания не влияет на результат другого. Это свойство следует из определения геометрического распределения: количество испытаний до успеха не зависит от предыдущих успехов.


## Задача 7 (№17 из учебника)

Игральная кость бросается до тех пор, пока впервые не выпадет меньше пяти очков. Обозначим x число очков, выпавших при последнем бросании игральной кости, и через y - число бросаний кости. Найти совместное распределение x и y. Являются ли x и y независимыми?

Здесь x заменяет Θ, y - заменяет ν

### Решение:

▎Распределение Y

Сначала определим распределение  Y . Вероятность того, что первый бросок будет неудачным (выпадет 5 или 6), равна:
- ``` P(неудачный бросок) = P(X = 5) + P(X = 6) = 1 / 6 + 1 / 6 = 1 / 3 ```

Вероятность того, что первый бросок будет успешным (выпадет 1, 2, 3 или 4):
- ``` P(успешный бросок) = P(X < 5) = 4 / 6 = 2 / 3 ```


Таким образом, случайная величина  Y  подчиняется геометрическому распределению с параметром  p = 2/3:
- ``` P(Y = k) = (( 1 / 3 ))ᵏ⁻¹ (( 2 / 3 )),   k = 1, 2, 3, … ```


▎Совместное распределение (X, Y)

Теперь найдем совместное распределение  (X, Y) . Условно на каждое значение  Y = k :
- Если  Y = k , это означает, что первые  k-1  бросков были неудачными (выпадали 5 или 6), а последний бросок (который равен  X ) был успешным (выпало одно из значений 1, 2, 3 или 4).

Таким образом, вероятность того, что при  Y = k  выпадет конкретное значение  X = x  (где  x = 1, 2, 3, 4 ):
- ``` P(X = x | Y = k) = P(первый k-1 бросков: 5 или 6) ⋅ P(X = x) ```


Вероятность того, что первые  k-1  бросков неудачны:
- ``` P(Y = k) = (( 1 / 3 ))ᵏ⁻¹ ⋅ 2 / 3 ```


Следовательно, ``` P(X = x | Y = k) = P(X = x) = 1 / 4,   x = 1, 2, 3, 4 ```

Итак, ``` P(X = x, Y = k) = P(Y = k) P(X = x | Y = k) = P(Y = k) P(X = x) = (( 1 / 3 ))ᵏ⁻¹ ⋅ 2 / 3 ⋅ 1 / 4 = 1 / 6 ⋅ ( 1 / 3 )ᵏ⁻¹ ```

▎Независимость

Теперь проверим независимость случайных величин  X  и  Y . Если они независимы, то должно выполняться:
```
P(X = x, Y = k) = P(X = x) P(Y = k)
```

Мы уже нашли:
- P(X = x) = 1/4 
- P(Y = k) = (( 1/3 ))ᵏ⁻¹ ⋅ 2/3 

Таким образом,
```
P(X = x) P(Y = k) = 1 / 6 ⋅ (( 1 / 3 ))ᵏ⁻¹
```
Это совпадает с ранее найденным совместным распределением:
```
P(X = x, Y = k) = P(X=x|Y=k) P(Y=k)
```

Таким образом, случайные величины  X  и  Y  независимы. 

## Задача 8 (№18 из учебника)

На n станках одновременно началась обработка n деталей. Предполагая, что времена обработки деталей независимы и имеют показательные распределения с параметром α, найти распределение времени:
а) до получения первой обработанной детали;
б) до окончания обработки всех деталей

### Решение:

Чтобы решить эту задачу, давайте рассмотрим каждую часть отдельно.

▎a) Время до получения первой обработанной детали

Если у нас есть  n  станков, и каждый из них обрабатывает деталь с временем обработки, распределенным по показательному закону с параметром  α , то время обработки каждой детали будет независимым и идентично распределенным (iid) случайным величинам.

Показательное распределение с параметром  α  имеет функцию плотности вероятности:
```
f(t) = α e⁻ᵅ ᵗ   (t ≥ 0)
```

Время до получения первой обработанной детали будет равно минимальному времени обработки среди  n  деталей. Обозначим времена обработки как  T₁, T₂, …, Tₙ . Тогда:
```
Tₘᵢₙ = min(T₁, T₂, …, Tₙ)
```

Согласно свойству минимального значения для независимых показательных распределений, если  Tᵢ ∼ Exp(α) , то:
```
Tₘᵢₙ ∼ Exp(nα)
```


▎b) Время до окончания обработки всех деталей

Время до окончания обработки всех деталей будет равно максимальному времени обработки среди  n  деталей. Обозначим его как: ``` Tₘₐₓ = max(T₁, T₂, …, Tₙ) ```


Для нахождения распределения максимума необходимо учитывать, что ``` Tᵢ ∼ Exp(α) ```. Время до завершения всех обработок можно выразить через распределение максимума:

Для  n  независимых случайных величин  Tᵢ , распределение их максимума можно найти следующим образом:

1. Найдем функцию распределения  `` F_(Tₘₐₓ)(t) ```:
```
F_Tₘₐₓ(t) = P(Tₘₐₓ ≤ t) = P(T₁ ≤ t, T₂ ≤ t, …, Tₙ ≤ t) = P(T₁ ≤ t)ⁿ
```

2. Поскольку  ``` P(Tᵢ ≤ t) = 1 - e⁻ᵅ ᵗ ```, мы имеем:
```
F_Tₘₐₓ(t) = (1 - e⁻ᵅ ᵗ)ⁿ
```

3. Чтобы найти функцию плотности вероятности  ``` f_Tₘₐₓ(t) ```, нужно продифференцировать функцию распределения:
```
f_Tₘₐₓ(t) = d / dt F_Tₘₐₓ(t) = n(1 - e⁻ᵅ ᵗ)ⁿ⁻¹ ⋅ (α e⁻ᵅ ᵗ)
```

Таким образом, время до окончания обработки всех деталей имеет плотность вероятности:
```
f_Tₘₐₓ(t) = nα e⁻ᵅ ᵗ (1 - e⁻ᵅ ᵗ)ⁿ⁻¹
```

▎Итоги

• a) Время до получения первой обработанной детали: `` Tₘᵢₙ ∼ Exp(nα) ```

• b) Время до окончания обработки всех деталей: имеет плотность вероятности ``` f_Tₘₐₓ(t) = nα e⁻ᵅ ᵗ (1 - e⁻ᵅ ᵗ)ⁿ⁻¹ ```

## Задача 9 (№20 из учебника)

Машина состоит из 10000 деталей. Каждая деталь независимо от других оказывается неисправной с вероятностью pi, причём для n1 = 1000 деталей p1 = 0,0003; для n2 = 200 деталей p2 = 0,0005 и для n3 = 7000 деталей p3 = 0,0001. Машина не работает, если в ней неисправны хотя бы две детали. найти приближенное значение вероятности того, что машина не будет работать. Использовать теорему пуассона

### Решение:

Для решения задачи о вероятности того, что машина не будет работать, воспользуемся теорией вероятностей и приближением по распределению Пуассона.

▎Шаг 1: Определение параметров

У нас есть три группы деталей с различными вероятностями неисправности:
1. Для  n₁ = 1000  деталей, вероятность неисправности  p₁ = 0.0003 .
2. Для  n₂ = 200  деталей, вероятность неисправности  p₂ = 0.0005 .
3. Для  n₃ = 7000  деталей, вероятность неисправности  p₃ = 0.0001 .

Общая вероятность неисправности для каждой группы:

1. Для первой группы: ``` λ₁ = n₁ ⋅ p₁ = 1000 ⋅ 0.0003 = 0.3 ```

2. Для второй группы: ``` λ₂ = n₂ ⋅ p₂ = 200 ⋅ 0.0005 = 0.1 ```

3. Для третьей группы: ``` λ₃ = n₃ ⋅ p₃ = 7000 ⋅ 0.0001 = 0.7 ```


Общая средняя величина неисправных деталей  λ :
```
λ = λ₁ + λ₂ + λ₃ = 0.3 + 0.1 + 0.7 = 1.1
```

▎Шаг 2: Применение распределения Пуассона

Согласно теореме Пуассона, количество неисправных деталей  X  можно аппроксимировать распределением Пуассона с параметром  λ = 1.1 :
```
P(X = k) = e^-λ λᵏ / k!
```

▎Шаг 3: Вероятность того, что машина не будет работать

Машина не будет работать, если неисправны хотя бы две детали, т.е. мы ищем  P(X ≥ 2) . Мы можем использовать дополнение:
```
P(X ≥ 2) = 1 - P(X = 0) - P(X = 1)
```

Теперь найдем  P(X = 0)  и  P(X = 1) :

- Вероятность того, что ни одна деталь не неисправна:
	```
	P(X = 0) = e^-1.1 (1.1)⁰ / 0! = e^(-1.1) ≈ 0.3329
	```
- Вероятность того, что одна деталь неисправна:
	```
	P(X = 1) = e^-1.1 (1.1)¹ / 1! = e^(-1.1) ⋅ 1.1 ≈ 0.3662
	```

▎Шаг 4: Подсчет вероятности

Теперь подставим значения в формулу:
```
P(X ≥ 2) = 1 - P(X = 0) - P(X = 1) ≈ 1 - 0.3329 - 0.3662 ≈ 1 - 0.6991 = 0.3009
```
## Задача 10 (№5 из учебника)

Точка P равномерно распределена на единичном квадрате ABCD. Найти плотность распределения площади ξ прямоугольника AB'PD', где B' и D' - основная перпендикуляров, опущенных из точки P на стороны AB и AD соответственно

### Решение:

▎Шаг 1: Определение координат

Обозначим координаты точки P как P(x, y), где x и y равномерно распределены на отрезке [0, 1].

▎Шаг 2: Определение координат точек B' и D'

- Точка B' — это проекция точки P на сторону AB, то есть ее координаты будут B'(x, 0).
- Точка D' — это проекция точки P на сторону AD, то есть ее координаты будут D'(0, y).

▎Шаг 3: Площадь прямоугольника AB'PD'

Площадь прямоугольника AB'PD' можно вычислить по формуле:
```
ξ = AB' ⋅ AD' = (y - 0)(x - 0) = xy
```

▎Шаг 4: Нахождение распределения площади ξ = xy

Теперь нам нужно найти распределение случайной величины ξ = XY. Поскольку X и Y независимы и равномерно распределены на интервале [0, 1], их плотности распределения равны:
```
f_X(x) = 1,   0 < x < 1
f_Y(y) = 1,   0 < y < 1
```

▎Шаг 5: Нахождение функции распределения для произведения

Чтобы найти плотность вероятности для произведения двух независимых случайных величин ξ = XY, воспользуемся методом преобразования. Для этого найдем функцию распределения F_\xi(s) = P(XY ≤ s):

F_ξ(s) = P(XY ≤ s) = ∫₀¹ P(X ≤ s / y) f_Y(y) dy


Поскольку Y равномерно распределен на [0, 1], имеем:
```
P(X ≤ s / y) =
1) 0 если y < s
2) s / y если y ≥ s
```

Таким образом, интеграл можно разбить на два случая:

1. Если s < 1:

	Интегрируем от s до 1:

	``` F_ξ(s) = ∫ₛ¹ s / y dy = s ∫ₛ¹ y⁻¹ dy = s [ln(y)]ₛ¹ = s (ln(1) - ln(s)) = -sln(s) ```


2. Если s > 1, то:

	Здесь вероятность равна 1, так как произведение двух величин не может превышать 1:

	``` F_ξ(s) = 1 ```


▎Шаг 6: Нахождение плотности вероятности

Теперь найдем плотность распределения ``` fξ(s) = (dFξ(s))/ds ```:

Для s < 1:

```
f_ξ(s) = -ln(s) - 1
```

Для s > 1:
```markdown
f_ξ(s) = 0
```

▎Итоговая плотность распределения

Итак, окончательная форма плотности распределения площади прямоугольника AB'PD':

```
f_ξ(s) =
1) -ln(s) - 1, 0 < s < 1
2) 0, s ≥ 1
```

Таким образом, мы нашли плотность распределения площади прямоугольника, образованного точкой, равномерно распределенной в единичном квадрате.
